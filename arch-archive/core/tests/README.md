# Core Tests

Test suite for Livnium Core System components.

## Contents

- **`test_livnium_core.py`**: Tests for the main LivniumCoreSystem
- **`test_quantum.py`**: Tests for quantum layer components
- **`test_qubit_capacity.py`**: Tests for qubit capacity calculations
- **`test_entanglement_capacity.py`**: Tests for entanglement capacity
- **`test_generalized_n.py`**: Tests for generalized N×N×N systems

## Purpose

This test suite verifies:
- Core geometry operations
- Quantum layer functionality
- Capacity calculations
- System correctness across different lattice sizes

Run tests to verify system integrity and capacity claims.
